# My Azure Website

This project demonstrates a Jenkins CI/CD pipeline deploying a static website to an Azure Virtual Machine.

## Project Files

- `index.html`: The main web page.
- `Jenkinsfile`: Defines the CI/CD pipeline.
- `README.md`: Project documentation.

## Deployment

The website is deployed to an Azure VM using Jenkins and SSH. After deployment, it can be accessed via the public IP of the VM.
